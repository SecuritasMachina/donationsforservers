# donationsforservers

A single-page static donation landing site ("Donations4Servers") titled *"Help those who serve on you"*, dedicated to helping restaurant and bar service workers affected by closures. The entire site is one static page, `WebContent/index.html`, which presents a short message and a PayPal donation button. It is packaged as an Eclipse Web Tools Platform (WTP) dynamic web project (see `.project` and `.settings/`) and styled with Bootstrap 4 loaded from a CDN; there is no server-side code, build pipeline, or backend.
